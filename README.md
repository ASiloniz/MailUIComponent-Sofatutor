# MailUIComponent-Sofatutor
MailUIComponent using React/Redux and the API Mailsac

# Getting Started:
To run the app please perform the following in the root folder:

## Installation:
`$ npm install`

## Running the app:
`$ yarn run dev-server`

After that you will see the app running in localhost:8080.
